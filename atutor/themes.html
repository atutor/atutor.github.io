Themes

A different look and feel can be applied to an ATutor systems using themes. Refer to the Theme Designer Documentation for more about creating your own custom theme. More information on installing themes can be found in the ATutor Administrator Handbook or in the themes_readme.txt file located in the /themes/ directory of your ATutor installation.

Submit Theme

Share your creativity with the ATutor community by submitting a theme exported from your ATutor installation. You must be signed-in to contribute themes.

Share your ATutor Theme
Submit a Theme!

Theme Gallery

Select the link on a theme's thumbnail image to preview the theme in a new window, and select the "Download" link to download the theme itself. Or, use the Administrator's Theme Manager to import themes directory from atutor.ca.

Note: These themes are for the ATutor 1.6 series. There is not a direct upgrade path from 1.6 to 2.0, though in some cases it may be possible to start with a copy of the ATutor 2.0 default theme (or the essential theme files as described in the Theme Designer Documentation), and to take the stylesheet from an older theme and copy it into the styles.css file of your new theme
